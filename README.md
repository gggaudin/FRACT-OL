# FRACT'OL//42PARIS

Fract'ol is a project that consists in graphically creating fractals.
The programm shows you 3 fractals and a DigAndClick Game.

## Usage :

```
./fractol [ma] MANDELBROT
		  [ju] JULIA
		  [cel] CELTIC
		  [bat] MiniGame
```

### Key :
	
	SCROLL == zoom in/out
	R = reset
	1 2 3 (on NumPad) == change colors
	F == 3D Launcher
	SPACEBAR == freeze
	ESC == quit
